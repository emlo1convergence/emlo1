About this project

This project is aimed to help Pytorch machine learning developers to quickly build a Flask web app in a Docker container ready to be deployed.
View on GitHub

By default, this app uses MobileNetV2 image classifier that was pre-trained on the ImageNet dataset. This application allows to upload upto maximum of 3 images simultaneously and infer their results. The Classification Name and Confidence level of the last 5 images uploaded are displayed. More explanation about this project can be found in the GitHub repository link.
